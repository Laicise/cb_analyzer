# 可转债分析项目分析报告

## 一、项目目的

这是一个**可转债（Convertible Bond）首日开盘价预测系统**，基于机器学习技术，帮助投资者预测新上市可转债的开盘价格。

### 核心功能
- **ML模型v6**：LightGBM分位数回归 + Ridge元学习器Stacking
- **相似度匹配**：基于ML学习权重的历史相似债券匹配
- **智能组合**：90% ML + 10% 相似度动态加权
- **数据获取**：自动从AKShare和东方财富获取实时数据
- **桌面GUI**：提供Linux桌面图形界面

## 三、功能流程

```
数据获取 → 特征工程 → 模型预测/相似度匹配 → 智能组合 → 输出结果
    ↓           ↓              ↓              ↓
 fetch_*   28维特征    ml_model_v6    combine_predictions
```

1. **数据获取**：从AKShare获取可转债、正股基本信息、PE/PB等
2. **特征工程**：提取28维特征（转股价值、溢价率、PE、PB、行业、市场情绪等）
3. **预测输出**：
   - ML模型v6预测（LightGBM分位数回归 + Ridge Stacking）
   - 相似度匹配（ML学习的最优权重）
   - 智能组合给出最终预测和置信区间

## 四、模型改进历程

### v5 → v6 关键改进
1. **LightGBM分位数回归**：替代sklearn GBDT，更精准的区间预测
2. **Ridge元学习器**：替代简单平均的Stacking集成
3. **市场情绪因子**：基于历史批次数据的市场热度和情绪
4. **新特征**：纯债价值、转股/规模比、小盘债标记、PBV/规模比

### v6性能提升

|------|-----|-----|------|
| Stacking MAE | 7.26元 | 5.24元 | +28% |
| R² | 0.2274 | 0.3145 | +38% |
| 预测区间覆盖率 | N/A | 47.2% | 新增 |

## 五、局限性

### 1. 数据层面
- ⚠️ **依赖外部API**：AKShare/东方财富可能有访问限制或数据延迟
- ⚠️ **数据质量**：部分基本面数据可能缺失或不准确
- ⚠️ **样本量有限**：约360条历史样本，模型泛化能力受限

### 2. 模型层面
- ⚠️ **MAE=5.24元**：平均误差约5.24元，对于100元面值的债券误差约5%
- ⚠️ **R²=0.31**：仅能解释约31%的价格 variance
- ⚠️ **市场时机异常**：极端误差来自市场情绪异常（面值开盘），模型无法预测

### 3. 工程层面
- ⚠️ **缺少测试**：无单元测试和集成测试
- ⚠️ **无日志系统**：难以追踪问题和优化
- ⚠️ **部署复杂**：依赖桌面环境，Linux only

### 4. 业务层面
- ⚠️ **仅预测首日**：未覆盖上市后交易决策
- ⚠️ **无风控模块**：无法设置止损/止盈
