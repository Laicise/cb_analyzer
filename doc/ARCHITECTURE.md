# 可转债分析器 - 架构设计文档

## 1. 系统概述

**项目名称**: CB Analyzer  
**项目类型**: 金融数据分析工具 + 桌面应用  
**核心功能**: 基于机器学习的可转债首日开盘价预测系统

## 3. 模块设计

### 3.1 核心模块

| 模块 | 职责 | 关键类/函数 |
|------|------|------------|
| `main.py` | CLI入口，命令分发 | `init()`, `update_data()`, `predict_bond()`, `train_model()`, `backtest()` |
| `desktop_app.py` | GUI应用，交互界面 | `CBAnalyzerGUI` 类，预测面板，结果展示 |
| `config.py` | 配置管理 | 数据库路径、评级映射、参数配置 |

### 3.2 预测引擎

| 算法 | 文件 | 说明 |
|------|------|------|
| 相似度匹配 | `similarity*.py` | 基于正股基本面的历史相似债券匹配 |
| ML集成模型 | `ml_model_v5.py` | 线性回归 + K近邻 + 梯度提升集成 |
| 特征工程 | `fundamental_features.py` | 17维特征：PE、PB、ROE、市值、行业等 |

### 3.3 数据模型

```python
# 核心数据模型
BondInfo {
    bond_code: str          # 债券代码
    bond_name: str         # 债券名称
    stock_code: str        # 正股代码
    conversion_value: float # 转股价值
    premium_rate: float    # 转股溢价率
    first_open: float      # 首日开盘价（目标变量）
    # ... 25+ 字段
}

StockInfo {
    stock_code: str        # 股票代码
    pe_ttm: float          # 市盈率TTM
    pb: float             # 市净率
    roe: float            # 净资产收益率
    market_cap: float     # 总市值
    industry_sw_l1: str   # 申万行业
    # ... 20+ 字段
}
```

## 4. 数据流

```
用户输入(债券代码)
       │
       ▼
┌──────────────────┐
│  desktop_app.py  │  或  main.py predict
└──────────────────┘
       │
       ▼
┌──────────────────────────────────────┐
│        特征提取 (17维)                │
│  - 目标债券: 转股价值、溢价率、评级   │
│  - 正股基本面: PE、PB、ROE、市值      │
│  - 市场情绪: 同日转债涨跌幅           │
└──────────────────────────────────────┘
       │
       ▼
┌──────────────────────────────────────┐
│  并行计算:                           │
│  1. 相似度匹配 → 历史相似债券        │
│  2. ML模型预测 → 价格区间            │
│  3. 综合给出最终预测                  │
└──────────────────────────────────────┘
       │
       ▼
┌──────────────────┐
│  结果展示/存储   │
└──────────────────┘
```

## 6. 部署架构

### 开发环境
```bash
pip3 install -r requirements.txt
python3 main.py init
python3 main.py update
```

### Linux桌面版
```bash
chmod +x run.sh
./run.sh
# 或
python3 desktop_app.py
```

### 打包发布
```bash
pyinstaller CBpredictor.spec
# 输出: dist/CBpredictor
```
