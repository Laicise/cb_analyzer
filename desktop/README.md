# 可转债预测工具 Linux 桌面版

## 概述

基于 PyQt6/PyInstaller 的可转债价格预测桌面应用，支持 Linux/macOS/Windows。

## 功能

- 6位可转债代码输入
- 一键预测首日开盘价
- 三个子模型加权平均
- 预测可信区间显示
- 实际开盘价对比

## 运行方式

### 直接运行（需要Python环境）
```bash
cd ~/Documents/openclaw/workspace_tony/code/cb_analyzer
python3 desktop_app.py
```

### 打包后运行
```bash
./dist/cb_predictor
```

## 系统要求

- Python 3.10+
- tkinter (通常随Python预装)
- Linux: GTK3 库

## 打包命令

```bash
pyinstaller --name CBpredictor \
            --onefile \
            --windowed \
            --add-data "db:db" \
            --add-data "analysis:analysis" \
            --add-data "config.py:." \
            --add-data "models:models" \
            --hidden-import=akshare \
            --hidden-import=numpy \
            --hidden-import=sqlalchemy \
            desktop_app.py
```
